# 📸 Instagram CLI Simulator (Java)

A simple command-line based Instagram simulation built using Java.
This project demonstrates core Object-Oriented Programming concepts along with data structures like HashMap, LinkedHashMap, and LinkedList.

---

## 🚀 Features

* 👤 Add Users
* 📝 Create Posts
* 🌍 View All Public Posts
* 🔍 View Posts by Specific User
* ❤️ Like Posts
* 💬 Comment on Posts with Timestamp

---

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)
* Encapsulation
* Java Collections Framework:

  * HashMap
  * LinkedHashMap
  * LinkedList
* CLI-based interaction using Scanner
* Basic system design logic

---

## 🏗️ Project Structure

```
Instagram/
│
├── Instagram.java          # Main class (entry point)
├── Instagram_Manager.java  # Core logic handler
├── User.java               # User model
├── Post.java               # Post model
├── Comment.java            # Comment model
```

---

## ⚙️ How It Works

1. Start the program
2. Add users
3. Create posts using user ID
4. View all posts
5. Like or comment on posts
6. View posts by a specific user

---

## ▶️ Sample Menu

```
1. Add User
2. Post
3. View Public Post
4. Post By User
5. Exit
```

---

## 🧩 Example Functional Flow

* Create user → Assign ID
* Create post → Linked to user
* View posts → Shows likes & comments
* Add like/comment → Updates dynamically

---

## 📂 Code Highlights

* Comment stores timestamp using LocalTime 
* Posts maintain likes and comments list 
* Manager class handles all operations centrally
